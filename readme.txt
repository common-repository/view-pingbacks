== Description ==

This plugin allows us to generate a list of pingbacks in delimited text format so we can use them for miscellaneous purpouses. 

This plugin is developed & maintained by the <a href="http://www.hatnohat.com">Hatnohat Marketing & Development Team</a>, 
which also provides commercial <a href='http://www.blogsense-wp.com/'>Blog Automation Software</a> and <a href='http://www.wptraffictools.com'>Traffic Monetization Plugins</a> for Wordpress Bloggers.

== Installation ==

1. Upload `view-pingbacks` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
